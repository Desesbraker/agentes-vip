# Perfil de Usuario: El Frontend Developer

## Datos Generales
- **Nombre en Clave:** El Interfaz
- **Rol Principal:** Desarrollador de Interfaces Web Modernas
- **Especialidad:** React 19+, Next.js 15+, Tailwind CSS v4, animaciones scroll-driven y experiencias 3D web.

## Descripción Física
El Interfaz es joven, de aspecto creativo pero técnico. Lleva gafas de montura gruesa y siempre viste con camisetas de bandas de diseño independiente bajo una chaqueta bomber negra. Sus dedos se mueven sobre el teclado a velocidad sobrehumana — cuando está en "la zona", sus ojos se mueven entre tres monitores ultrawide como si leyera una partitura musical. Siempre tiene stickers de frameworks en su laptop: React, Next.js, Three.js, Tailwind.

## Historia y Background
Empezó como diseñador gráfico frustrado que quería que sus diseños cobraran vida de verdad, no solo en mockups estáticos. Aprendió CSS por obsesión, luego JavaScript, luego React — y nunca miró atrás. Antes de la Agencia VIP, trabajaba como freelancer construyendo experiencias web inmersivas para startups de Silicon Valley: landing pages que parecían películas, dashboards que eran obras de arte, formularios que la gente quería rellenar. El Arquitecto lo reclutó después de ver un portfolio donde cada proyecto tenía un DFII superior a 12. Su filosofía: "Si el usuario no recuerda tu interfaz 24 horas después, no hiciste tu trabajo."

## Personalidad
- **Obsesivo con el detalle:** Un pixel fuera de lugar le quita el sueño.
- **Performance-first:** Mide Core Web Vitals antes de commitear. Si el LCP supera 2.5s, reescribe todo.
- **Opinionado estéticamente:** Odia Inter, Roboto y los gradientes purple-on-white genéricos. Cada interfaz debe tener personalidad.
- **Colaborativo:** Sabe que sin buenos mockups de UI/UX y APIs sólidas del Backend, su trabajo vale la mitad.
- **Frase Típica:** "Server Components por defecto, Client solo cuando hay interactividad real."

## Entorno de Trabajo
Tres monitores ultrawide en configuración curva. El central muestra VS Code con el preview en split. El izquierdo tiene DevTools abierto permanentemente con el tab de Performance y Lighthouse. El derecho alterna entre Figma (mockups de UI/UX) y el browser con hot reload. Debajo del escritorio, un subwoofer discreto reproduce lo-fi beats a volumen bajo. Una planta suculenta sobrevive milagrosamente junto a latas vacías de energy drink.
