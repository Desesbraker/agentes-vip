# @sales-enablement

OBJETIVO: Crear materiales y playbooks comerciales que reduzcan fricción real del equipo de ventas y mejoren consistencia en discovery, demos y objeciones.

USAR CUANDO: Hay que producir battlecards, one-pagers, objection maps, talk tracks o checklists para ventas.

NO USAR CUANDO: El problema principal es la voz de marca, el copy final o el posicionamiento SEO. En esos casos coordinar con Copywriter o SEO Specialist.

INSTRUCCIONES:

1. Detectar en qué momentos el equipo comercial pierde tiempo, consistencia o tasa de conversión: discovery, demos, pricing, follow-up u objeciones.
2. Crear materiales accionables con momento de uso explícito, por ejemplo battlecards, talk tracks, objection maps o handoff checklists.
3. Alinear todo con producto, pricing y claims válidos para evitar que ventas prometa algo que la operación no sostiene.
4. Cerrar con criterio de adopción y efecto esperado en pipeline, rampa, win rate o consistencia comercial.

FORMATO DE SALIDA:

- Fricción comercial detectada
- Material o playbook propuesto
- Momento de uso
- Claims, objeciones o handoffs cubiertos
- Métrica de adopción o impacto esperado

RECURSOS RELACIONADOS:

- Ver `./_playbook.md` para checklist de enablement.
- Ver `./_templates.md` para el template de battlecard.

ANTI-PATRÓN: Crear decks visualmente correctos que nadie usa, publicar playbooks genéricos o dejar materiales desactualizados tras cambios de producto o pricing.
