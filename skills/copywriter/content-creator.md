# @content-creator

OBJETIVO: Diseñar y redactar contenido editorial de marca para blog, social media y calendarios de contenido con una voz consistente y una estructura reutilizable por canal.

USAR CUANDO: Hay que crear contenido de marca, piezas SEO editoriales, contenido social o un calendario mensual de publicaciones.

NO USAR CUANDO: La tarea principal es escribir copy de conversión para una landing, anuncio o página de producto. En ese caso usar @copywriting.

INSTRUCCIONES:

1. Definir la voz de marca con arquetipo narrativo y 3-5 atributos de tono verificables, por ejemplo claro, experto, cercano o desafiante según el proyecto.
2. Para contenido SEO, validar keyword principal, intención de búsqueda, estructura y preguntas clave. Priorizar cobertura útil y profundidad; no usar keyword density como métrica de calidad.
3. Para social media, adaptar la idea central por plataforma usando una repurposing matrix con formato, longitud, hook y CTA por canal.
4. Planificar el calendario con pilares explícitos. Por defecto: 40% educativo, 25% autoridad o prueba, 25% promocional y 10% comunidad o cultura, salvo que el proyecto defina otra mezcla.

FORMATO DE SALIDA:

- Objetivo editorial y audiencia
- Voz de marca y pilares de contenido
- Outline o calendar de piezas
- Keyword, intención y CTA por pieza cuando aplique
- Riesgos de canibalización, inconsistencia o falta de foco

RECURSOS RELACIONADOS:

- Ver `./_playbook.md` para glosario, repurposing matrix y checklist editorial.
- Ver `./_templates.md` para el outline de contenido y el calendario mensual.

ANTI-PATRÓN: Escribir antes de validar tema o keyword, hacer keyword stuffing, cambiar la voz de marca entre piezas o publicar contenido sin CTA ni criterio de distribución.
