# @canvas-design

## OBJETIVO

Diseñar piezas visuales o editoriales con intención formal fuerte, composición cuidada y calidad suficiente para presentaciones de alto valor estético.

## USAR CUANDO

Hay que producir una pieza gráfica de autor, un póster, una portada o una composición donde la dirección visual sea el entregable principal.

## NO USAR CUANDO

Lo que se necesita es una interfaz funcional, una landing o un diseño fuertemente utilitario. Esta skill prioriza obra visual y composición.

## INSTRUCCIONES

1. Desarrollar primero la filosofía visual, la referencia conceptual y la restricción formal que dará identidad a la pieza.
2. Traducir esa dirección a una composición de canvas con jerarquía, ritmo espacial, paleta contenida y uso deliberado del texto como forma.
3. Refinar balance, contraste, bordes y relaciones de escala hasta que la pieza se sostenga sin explicaciones adicionales.
4. Entregar el canvas final y una síntesis breve de la intención, dejando claro cómo se debe exportar o presentar.

## FORMATO DE SALIDA

- Intención o referencia visual
- Paleta y sistema formal
- Composición y jerarquía
- Entregables y formato
- Riesgo de caer en genérico

## ANTI-PATRON

Plantillas genéricas, texto excesivo, decoración infantil o elementos descontrolados que rompen el canvas y la dirección visual.
