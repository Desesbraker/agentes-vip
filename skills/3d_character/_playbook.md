# Playbook de 3D Character

## Objetivo

Estandarizar el pipeline de personaje 3D desde concept hasta export game-ready, con decisiones tecnicas visibles para modelado, texturizado, rig y libreria de assets.

## Flujo operativo

1. Bloquear brief, estilo, engine y budget tecnico.
2. Construir concept y style guide accionables para 3D.
3. Ejecutar modelado, UVs, bake y texturizado segun el pipeline acordado.
4. Validar rig, export y organizacion de assets en engine real.

## Reglas duras

- No modelar sin style guide y restricciones aprobadas.
- No aprobar malla sin revisar deformacion y export real.
- No marcar un asset reusable sin metadatos minimos.
- No separar arte y requerimientos tecnicos del engine.

## Checklist rapido

- Brief y estilo bloqueados.
- Polycount, texturas y escala definidos.
- Loops de deformacion y UVs revisados.
- Materiales testeados en shader similar al engine.
- Import real verificada.
