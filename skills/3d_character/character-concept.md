# @character-concept

## OBJETIVO

Crear un concept y style guide de personaje que sean visualmente utiles y tecnicamente producibles para 3D. La skill bloquea decisiones de silueta, materiales, escala y restricciones antes del modelado.

## USAR CUANDO

- Se necesita definir un personaje nuevo para juego, avatar o mascota.
- Hace falta traducir un brief visual a una hoja tecnica util para 3D.
- El pipeline aun no tiene style guide aprobado.

## NO USAR CUANDO

- El concept ya esta aprobado y toca modelar; usar 3d-modeling-pipeline.
- El trabajo es puramente UI o branding fuera de personaje 3D.
- Faltan estilo, referencias o engine objetivo.

## INSTRUCCIONES

1. Bloquear brief, uso del personaje, tono, referencias, silueta, proporciones y engine destino.
2. Definir vistas, materiales dominantes, accesorios, zonas rigidas o flexibles y limites de complejidad.
3. Traducir el estilo a decisiones de produccion: polycount, texel density, piezas intercambiables y blend shapes si aplican.
4. Entregar style guide accionable con riesgos de produccion y piezas obligatorias antes de pasar a modelado.

## FORMATO DE SALIDA

- Resumen del brief y estilo aprobado.
- Hoja tecnica del personaje.
- Restricciones de produccion y engine.
- Riesgos de pipeline.
- Criterio de paso a modelado.

## ANTI-PATRON

- Aprobar concepts bonitos pero no producibles.
- Cambiar proporciones a mitad del pipeline.
- Diseñar sin silueta, escala ni restricciones del engine.
