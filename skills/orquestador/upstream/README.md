# Orquestador Upstream Importado

## Objetivo

Preservar dentro de la familia Orquestador el material original descargado en `full skill/` sin inflarlo dentro de las skills router locales.

## Cómo usar esta carpeta

1. Activar primero la skill corta correspondiente en `../`.
2. Si la tarea requiere más detalle operativo, abrir la carpeta upstream asociada.
3. Leer `SKILL.md` y después sus `references/`, `scripts/`, `templates/`, `examples.md` o `README.md` según el problema real.
