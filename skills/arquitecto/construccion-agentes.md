# @construccion-agentes

## OBJETIVO

Construir perfiles nuevos de agente con una estructura consistente, enrutable y verificable antes de entrar en detalles de redaccion. La skill obliga a resolver inputs criticos y templates antes de declarar el perfil listo.

## USAR CUANDO

- Se necesita crear un perfil nuevo para un dominio o plataforma.
- Hay que traducir objetivos, outputs y boundaries a una estructura de agente estable.
- El sistema requiere un quality gate arquitectonico antes de redactar.

## NO USAR CUANDO

- Solo se va a auditar un perfil existente; usar auditoria-perfiles.
- Falta informacion critica y aun no se puede entrevistar o clasificar el agente.
- El trabajo es sobre una skill puntual, no sobre el perfil completo.

## INSTRUCCIONES

1. Ejecutar entrevista base, clasificar el dominio y mapear outputs primarios del agente.
2. Completar el inventario estructural obligatorio antes de redactar el perfil.
3. Redactar las secciones requeridas con reglas claras de routing, outputs y boundaries.
4. Validar con checklist y crear templates faltantes antes de entregar la version final.

## FORMATO DE SALIDA

- Dominio y objetivo del agente.
- Estructura obligatoria completada.
- Outputs y templates requeridos.
- Riesgos o respuestas faltantes.
- Checklist final de validacion.
