## Objetivo

Preservar dentro de la familia Arquitecto el material original descargado en `full skill/` sin inflarlo dentro de las skills router locales.

## Cómo usar esta carpeta

1. Activar primero la skill corta correspondiente en `../`.
2. Si la tarea requiere más detalle metodológico, abrir la carpeta upstream asociada.
3. Leer `SKILL.md` y después sus archivos auxiliares según el problema real.
