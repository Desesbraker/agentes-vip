# Resources: Backend

Guía operativa extendida para la familia Backend.

Estos recursos recuperan profundidad del upstream de Antigravity sin inflar los archivos principales de skill.

## Mapa de uso

- `backend-nodejs-playbook.md`
  - Usar con `@backend-dev-guidelines` y `@nodejs-best-practices`
- `api-design-playbook.md`
  - Usar con `@api-patterns` y `@api-design-principles`
- `database-postgres-playbook.md`
  - Usar con `@database-design`, `@sql-pro` y `@postgres-best-practices`
- `python-frameworks-playbook.md`
  - Usar con `@python-pro`, `@fastapi-pro` y `@django-pro`
- `stripe-billing-playbook.md`
  - Usar con `@stripe-integration`

## Regla operativa

1. La skill principal sigue siendo el router corto.
2. Si la tarea exige implementación real, ejemplos, checklists o scaffolding, abrir el playbook temático correspondiente.
3. Si el playbook no alcanza, escalar el gap explícitamente en vez de improvisar.
