# Backend Upstream Importado

## Objetivo

Preservar dentro de la familia Backend el material original descargado en `full skill/` sin comprimirlo dentro de las skills router locales.

## Cómo usar esta carpeta

1. Activar primero la skill corta correspondiente en `../`.
2. Si la tarea requiere más detalle operativo, abrir la carpeta upstream asociada.
3. Leer `SKILL.md` y después sus `resources/`, `references/`, `assets/`, `scripts/` o archivos temáticos según el problema real.

## Regla de fidelidad

Estos contenidos se copian desde la carpeta local `full skill/` para conservar texto, estructura y anexos del origen. Solo deben ajustarse rutas o notas de integración cuando sea necesario para el sistema local.

## Nota de origen

La mayoría de carpetas importadas provienen de `full skill/skills/`. `postgres-best-practices/` se importó desde `full skill/plugins/antigravity-awesome-skills/skills/` porque en la descarga local esa es la variante completa con reglas, README, metadata y documento compilado.
