# @video-repurposing

OBJETIVO: Convertir contenido largo en clips útiles para distintos canales sin perder contexto, intención ni capacidad de reutilización.

USAR CUANDO: Hay que extraer clips o piezas derivadas de webinars, entrevistas, cursos, podcasts u otros formatos largos.

NO USAR CUANDO: El contenido fuente no tiene momentos autónomos o no existe un objetivo claro por plataforma. En ese caso no conviene fragmentar por fragmentar.

INSTRUCCIONES:

1. Analizar el contenido fuente y detectar momentos con autonomía real como hooks, demostraciones, objeciones, insights o frases memorables.
2. Reencuadrar cada clip para la nueva plataforma añadiendo contexto mínimo, apertura fuerte, overlays, ritmo y subtítulos cuando haga falta.
3. Crear variantes por objetivo, como awareness, autoridad, educación o conversión, para que el repurposing sea reinterpretación y no solo recorte.
4. Cerrar con export y metadata clara de tema, plataforma, CTA y biblioteca para reutilización futura.

FORMATO DE SALIDA:

- Fuente y clips seleccionados
- Plataforma y objetivo por clip
- Contexto mínimo y hook
- CTA y metadata
- Riesgos de pérdida de sentido o repetición

RECURSOS RELACIONADOS:

- Ver `./_playbook.md` para checklist de repurposing.
- Ver `./_templates.md` para la repurposing matrix.

ANTI-PATRÓN: Cortar por duración y no por idea, publicar clips sin contexto suficiente o convertir el repurposing en una fábrica de fragmentos sin estrategia ni biblioteca.
