# @subtitles-transcripts

OBJETIVO: Generar subtítulos, captions y transcripts legibles, sincronizados y útiles para accesibilidad, archivo o reutilización editorial.

USAR CUANDO: Hay que subtitular un video, producir transcript o revisar accesibilidad y legibilidad del texto sobre pantalla.

NO USAR CUANDO: El audio base está incompleto, roto o todavía no existe una versión de corte estable. Primero estabilizar la fuente.

INSTRUCCIONES:

1. Obtener una transcripción base limpia y corregir errores de nombres, tecnicismos, timings o reconocimiento automático antes de exportar.
2. Adaptar los subtítulos al formato con bloques legibles, tiempos cómodos, puntuación mínima útil y distinción de hablantes cuando corresponda.
3. Generar transcript completo cuando sirva para SEO, archivo o reutilización editorial, manteniendo consistencia con los captions del video.
4. Cerrar con validación de legibilidad sobre imagen, velocidad de lectura, sincronía y export final.

FORMATO DE SALIDA:

- Fuente y versión subtitulada
- Errores corregidos
- Criterios de legibilidad aplicados
- Transcript asociado
- Riesgos de sincronía o accesibilidad

RECURSOS RELACIONADOS:

- Ver `./_playbook.md` para checklist de subtítulos.
- Ver `./_templates.md` para el subtitle review.

ANTI-PATRÓN: Confiar ciegamente en auto-caption, usar líneas largas que tapan la imagen o entregar subtítulos sin revisar sincronía, contraste y legibilidad real.
