# QA Testing Upstream Importado

## Objetivo

Preservar dentro de la familia QA Testing el material original descargado en `full skill/` sin inflarlo dentro de las skills router locales.

## Cómo usar esta carpeta

1. Activar primero la skill corta correspondiente en `../`.
2. Si la tarea requiere más detalle operativo, abrir la carpeta upstream asociada.
3. Leer `SKILL.md` y después sus `resources/` o archivos temáticos según el problema real.
