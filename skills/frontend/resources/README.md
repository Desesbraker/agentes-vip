# Resources: Frontend

Guía operativa extendida para la familia Frontend.

La skill principal mantiene el routing corto. Estos recursos recuperan profundidad del upstream en arquitectura React, Next.js App Router, performance, Tailwind, formularios y motion.

## Mapa de uso

- `react-nextjs-playbook.md`
  - Usar con `@frontend-developer` y `@react-patterns`
- `performance-playbook.md`
  - Usar con `@react-best-practices`
- `nextjs-routing-cache-playbook.md`
  - Usar con `@nextjs-best-practices`
- `tailwind-design-system-playbook.md`
  - Usar con `@tailwind-patterns` y `@frontend-design`
- `forms-cro-playbook.md`
  - Usar con `@form-cro`
- `motion-immersive-playbook.md`
  - Usar con `@scroll-experience` y `@3d-web-experience`
- `typescript-frontend-playbook.md`
  - Usar con `@typescript-expert`

## Regla operativa

1. La skill corta define trigger, salida y guardrails.
2. El playbook correspondiente aporta workflow, quality gates y decisiones concretas.
3. Si la tarea exige arquitectura, caché, patterns o degradación cross-device, abrir el resource antes de codificar.
