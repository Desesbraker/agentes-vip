# Forms and CRO Playbook

## Cuándo abrir este recurso

- Formularios de captación, signup, checkout parcial o lead gen
- Necesidad de reducir fricción y aumentar conversión

## Flujo operativo

1. Diagnosticar fricción con datos o heurística explícita.
2. Revisar campos por valor de negocio.
3. Ordenar el flujo para reducir carga cognitiva.
4. Definir feedback, confianza y validación.
5. Testear una variable a la vez.

## Implementación recomendada

- React Hook Form + Zod cuando se necesite buen DX y validación robusta.
- Una columna por defecto.
- Labels visibles.
- Errores inline claros.
- CTA inequívoco.

## Checklist

- [ ] Objetivo de conversión claro
- [ ] Fricción principal identificada
- [ ] Campos cuestionados uno a uno
- [ ] Estado loading/success/error definido
- [ ] Hipótesis y métrica primaria definidas

## Entrega esperada

- Objetivo de conversión
- Fricción detectada
- Cambios prioritarios
- Hipótesis de impacto
- Métrica a validar
