# @market-sizing-analysis

## OBJETIVO

Estimar TAM, SAM y SOM con métodos triangulados y supuestos explícitos para evaluar oportunidades o soportar fundraising.

## USAR CUANDO

Hay que dimensionar un mercado nuevo, priorizar una categoría o defender una narrativa de oportunidad ante inversores o equipo interno.

## NO USAR CUANDO

Solo se busca un número decorativo para una slide sin intención de sostenerlo. Esta skill exige supuestos y fuentes auditables.

## INSTRUCCIONES

1. Definir segmento, geografía, producto y horizonte temporal antes de calcular cualquier cifra para no mezclar universos incompatibles.
2. Construir al menos un método top-down y uno bottom-up, dejando explícitos filtros, ARPU, volumen de clientes y señales comparables.
3. Triangular ambos métodos, explicar diferencias y usar benchmarks o comparables públicos para validar si el rango es defendible.
4. Cerrar con TAM, SAM y SOM como rango o escenario, no como número mágico, indicando riesgos y nivel de confianza.

## FORMATO DE SALIDA

- Segmento y alcance
- Método top-down
- Método bottom-up
- Rango TAM, SAM y SOM
- Supuestos, riesgos y confianza

## RECURSOS RELACIONADOS

- Ver `./_playbook.md` para checklist de triangulación.
- Ver `./_templates.md` para el template de market sizing.

## ANTI-PATRON

Usar solo top-down, confundir TAM con SAM, presentar SOM agresivo sin justificación, elegir datos convenientes o omitir fuentes y supuestos.
